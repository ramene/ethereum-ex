## Dependencies

1. Node +6.11
2. NPM +3.10
2. Truffle +4.0.5
3. Ganache CLI +6.0.4

## Clone

```
  git clone https://github.com/ramene/ethereum-ex.git
```

## Install

```
  cd ethereum-ex
  npm install
```

## Deploy

In another terminal start the ganache using the following command

```
  ganache-cli
```

In another terminal compile it, deploy and test

```
  truffle compile
  truffle migrate
  truffle test
```
